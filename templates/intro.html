Integrated Genome-Scale Metabolic Database

Developed for BF768: Biological Database Analysis, Spring 2025, at Boston University

Abstract

This database, built for the Segrè Lab, supports genome-scale metabolic modeling by integrating gap-filling, flux balance analysis (FBA), and version control. It stores draft models, annotations, and experimental data (e.g., growth conditions, reaction fluxes) for P. simiae, enabling researchers to identify missing reactions using KEGG and BiGG, visualize growth via COMETS simulations, and track model iterations. Designed for internal use, it offers a web interface for uploading models (XML, TSV), searching by media, and downloading data, with future potential for AI-driven model refinement.
