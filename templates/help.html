Help Guide

  1. Navigate to the Home/Search page to view recent model uploads or search by growth media.
  2. Use the "Upload New Model File" section to add new models (XML or TSV) and optional associated data files (TSV).
  3. Fill in metadata (like Growth Media, Annotation Tool) as applicable. Only the main model file is required.
  4. Click "Upload Model". You should see a success message, and the page will reload.
  5. Search results or the initial view will show a table of models.
  6. Click on any file link (e.g., under "Model File" or "Growth File (TSV)") to download the corresponding file.
  7. For further assistance or to report issues, contact: dukovski@bu.edu
